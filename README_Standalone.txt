DQA Dashboard - Standalone Application
=======================================

How to Use:
1. Copy the entire "DQA_Dashboard_Standalone" folder to any location
2. Run "DQA_Dashboard_Standalone.exe"
3. The application will automatically open in your default web browser
4. Use the application to view performance results

Features:
- No installation required
- Automatically finds available port
- Opens browser automatically
- All data included in the package
- No internet connection required

Notes:
- The application runs on http://127.0.0.1:XXXX (port automatically chosen)
- Analysis/upload features are disabled for end-users
- All existing data is included and viewable
- Close the application by closing the console window or browser tabs

Troubleshooting:
- If port is in use, restart the application
- If browser doesn't open, manually go to http://127.0.0.1:8000
- Make sure no other application is using the same port

System Requirements:
- Windows 7 or later
- No admin rights required
- 100MB free disk space
- Modern web browser (Chrome, Firefox, Edge)
